## 🎨 Design System

### Colors
```typescript
import { colors } from '@/design-system'

colors.primary.DEFAULT      // #8B5CF6 (customizable)
colors.secondary.DEFAULT    // #EC4899
colors.success.DEFAULT      // #10B981
colors.warning.DEFAULT      // #F59E0B
colors.error.DEFAULT        // #EF4444

colors.text.primary         // #1F2937
colors.text.secondary       // #6B7280
colors.surface.background   // #FFFFFF
colors.surface.elevated     // #F9FAFB
```

**Change once, update everywhere!**

### Typography
```typescript
import { typography } from '@/design-system'

typography.h1          // 32px, bold
typography.h2          // 24px, bold  
typography.body        // 16px, regular
typography.caption     // 12px, regular
typography.button      // 16px, semibold
```

### Spacing
```typescript
import { spacing } from '@/design-system'

spacing.xs     // 4px
spacing.sm     // 8px
spacing.md     // 16px
spacing.lg     // 24px
spacing.xl     // 32px
spacing['2xl'] // 48px
```

## 🚀 Deployment Options

### Instant Updates (Recommended)
```bash
eas update --branch production
```
- No app store review
- Users get updates instantly
- Perfect for bug fixes and features

### App Store Deployment
```bash
eas build --platform ios
eas submit --platform ios
```
- Full app submission
- First release and major updates

### Google Play Deployment
```bash
eas build --platform android
eas submit --platform android
```
